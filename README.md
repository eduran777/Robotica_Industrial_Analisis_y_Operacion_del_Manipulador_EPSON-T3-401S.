# Robotica_Industrial_Analisis_y_Operacion_del_Manipulador_EPSON-T3-401S.
Este repositorio documenta el trabajo con el manipulador EPSON T3-401S: comparación técnica con otros robots, configuración inicial, modos de operación, diseño de un gripper neumático y programación de una trayectoria que mueve dos huevos por una cubeta siguiendo el patrón del caballo en ajedrez.
